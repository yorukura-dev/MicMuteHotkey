# MicMuteHotkey

Windows 11 用のポータブルなマイクミュート切替ツールです。  
グローバルホットキーやタスクトレイ操作で、  
**Windows の既定の入力デバイス（マイク）を OS レベルでミュート／解除**できます。

## 特徴

- インストール不要（ポータブル）
- Windows 11 対応
- グローバルホットキーでミュート切替
- タスクトレイ常駐（状態をアイコンで表示）
- トレイアイコンのダブルクリックでも切替可能
- 設定ファイルは exe と同じフォルダに保存
- 通常ユーザー権限で動作
- レジストリ不使用

---

## 動作環境

- OS：Windows 11
- 権限：通常ユーザー権限

※ 本ソフトウェアは .NET ランタイムを内包しています。

## 使い方

### 起動
`MicMuteHotkey.exe` を起動すると、画面は表示されず  
タスクトレイに常駐します。

### ミュート切替

- 既定ホットキー：**Ctrl + Alt + M**
- ホットキーを押すたびにミュート／解除が切り替わります
- タスクトレイのアイコンを **ダブルクリック**しても切り替えられます
- 状態変更時に通知が表示されます

### トレイメニュー

タスクトレイのアイコンを右クリックすると以下の操作が可能です。

- 現在の状態表示（ミュート中 / ミュート解除）
- ホットキー設定
- アプリケーションの終了

---

## ホットキー設定

トレイメニューの「ホットキー設定」から変更できます。

- 入力欄をクリックしてからキー操作を行ってください
- 修飾キー単体（Ctrl のみ等）は指定できません
- 他のアプリとホットキーが競合している場合、登録に失敗します

---

## 設定ファイル

- 設定は `settings.json` として exe と同じフォルダに保存されます
- ファイルが存在しない場合はデフォルト設定で起動します
- exe と settings.json を一緒に移動すれば設定も引き継がれます

---

## 単一起動について

本ソフトウェアは同時に複数起動することはできません。  
すでに起動している場合、2回目以降の起動は行われません。

---

## アンインストール

インストール作業は行いません。  
以下のファイルを削除することで完全に削除できます。

- MicMuteHotkey.exe
- settings.json（存在する場合）

---

## 注意事項

- 操作対象は Windows の既定の入力デバイス（マイク）です
- 特定のマイクデバイスを固定指定する機能はありません
- アプリケーション単位のミュートは行いません
- 管理者権限が必要な機能はありません
- 自動アップデート機能はありません
